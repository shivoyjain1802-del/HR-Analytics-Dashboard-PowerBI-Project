# HR-Analytics-Dashboard-PowerBI-Project

Project Overview

This project is an HR Analytics Dashboard built using Power BI to analyze employee data and uncover insights related to workforce demographics, attrition, employee performance, and organizational trends. The dashboard provides an interactive and data-driven approach to help HR professionals make informed decisions and improve workforce management strategies.

 Objectives
Analyze employee demographics and workforce distribution.
Monitor employee attrition and retention trends.
Identify key factors influencing employee turnover.
Track department-wise and job-role-wise performance.
Provide actionable insights for HR decision-making.

 Tools & Technologies Used
Power BI Desktop
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions)
Data Modeling
Interactive Visualizations

Interactive KPI Cards

Key Insights
Identified departments and job roles with higher attrition rates.
Analyzed the relationship between employee age, experience, and turnover.
Determined the impact of overtime and salary on employee retention.
Highlighted demographic patterns influencing workforce distribution.

Business Recommendations
Implement retention strategies for high-attrition departments.
Review compensation structures for employee segments with higher turnover.
Introduce employee engagement programs for at-risk groups.
Monitor overtime practices to reduce employee burnout.
Focus on career development and training initiatives.
